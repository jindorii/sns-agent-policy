이 도구는 무엇을 하나요

sns-agent는 개발자 본인 소유의 인스타그램 계정에 새 릴스가 올라오면 이를 감지해 다운로드하고, 같은 사람이 소유한 유튜브 채널에 자동으로 업로드한 뒤 원본 게시물 링크를 담은 댓글을 남기는 개인용 자동화 스크립트입니다.

운영자, 사용자, 데이터 대상이 모두 동일한 한 사람(개발자 본인)이며, 별도의 회원가입이나 다중 사용자 기능은 존재하지 않습니다.

개인정보처리방침

수집·접근하는 데이터

연동 서비스접근 범위용도
Google / YouTube Data API youtube.upload, youtube.force-ssl 본인 소유 채널에 영상 업로드, 업로드한 영상에 첫 댓글 작성
Instagram Graph API 본인 계정의 게시물 조회(읽기 전용) 이미 게시된 본인 릴스의 영상 파일·캡션·링크 조회

하지 않는 것

  • 제3자에게 데이터를 판매하거나 공유하지 않습니다.
  • 광고, 프로파일링, 본 자동화 목적 외 어떤 용도로도 데이터를 사용하지 않습니다.
  • 개발자 본인 외 어떤 사람도 이 데이터에 접근하거나 열람하지 않습니다.
  • 사용자 계정을 별도로 만들거나 개인정보를 외부 서버에 전송하지 않습니다.

저장 및 보관

Google OAuth 인증 후 발급되는 refresh token은 개발자 본인이 소유·관리하는 서버(로컬 컴퓨터 또는 개인 VPS)의 파일에만 저장되며, 외부로 전송되거나 별도 데이터베이스·클라우드에 업로드되지 않습니다.

접근 철회 및 삭제

언제든지 Google 계정 설정 > 타사 액세스 권한에서 sns-agent의 접근 권한을 해제할 수 있습니다. 로컬에 저장된 토큰 파일을 삭제하면 즉시 접근이 중단됩니다.

이용약관

sns-agent는 일반 대중에게 제공되는 서비스가 아니라, 개발자 본인의 콘텐츠 자동화를 위해 만들어진 개인용 스크립트입니다. 별도의 가입 절차나 서비스 이용 계약이 존재하지 않습니다.

이 도구는 있는 그대로(as-is) 제공되며 어떠한 보증도 하지 않습니다. 유튜브·인스타그램 등 각 플랫폼의 자체 이용약관 및 API 이용 정책이 우선 적용됩니다.
